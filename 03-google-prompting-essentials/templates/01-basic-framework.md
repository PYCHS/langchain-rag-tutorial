# 01 — Basic Framework: Task + Persona + Format

The simplest useful prompt. Combines a **Task** with a **Persona** (role) and an **output Format**.

## Template

```
Act as {persona}.

{task}

Format the output as {format}.
```

## Example

```
Act as an anime expert.

Suggest gift ideas related to anime for my friend's birthday.

Format the output as a table with three columns: gift idea, genre, and price range.
```

## When to Use
- Quick, everyday tasks
- When you want results more specific than a default response
- Starting point before adding context and references
