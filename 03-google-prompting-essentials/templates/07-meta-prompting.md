# 07 — Meta Prompting

When you're stuck and don't know how to phrase your prompt, ask the AI to help you write it.

## Template

```
I want to use you to {high-level goal}, but I'm not sure how to write the best prompt for it.

Here's what I know about my situation:
- {detail 1}
- {detail 2}
- {detail 3}

Please write me an optimal prompt I can use to get the best result.
The prompt should include:
- A clear persona for you to take on
- All necessary context
- A specific task
- Output format

After writing the prompt, briefly explain why you structured it that way.
```

## Example

```
I want to use you to plan a 10-day Japan trip, but I'm not sure how to write the best prompt for it.

Here's what I know about my situation:
- Traveling with my partner in late October
- Total budget around $4,000 for both of us (excluding flights)
- We love food, history, and offbeat experiences; not big on shopping
- Energy level: medium — we don't want to be exhausted every day

Please write me an optimal prompt I can use to get the best result.
The prompt should include a clear persona, all necessary context, a specific task,
and output format. After writing the prompt, briefly explain why you structured it that way.
```

## When to Use
- Starting a new type of task
- When your prompts keep producing generic results
- Learning what good prompts look like for a domain you're new to
